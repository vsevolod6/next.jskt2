# Проект "Страны и города" на Next.js

Проект демонстрирует работу с многомерными структурами данных в Next.js, отображая информацию о странах и их городах.

## Функциональность

- **Главная страница**: Отображает список всех стран в виде ссылок
- **Страница страны**: Показывает подробную информацию о стране (язык, площадь, население) и список ее городов
- **Страница города**: Отображает информацию о городе (площадь, население, год основания) и ссылку на страницу страны

## Структура проекта

```
├── app/
│   ├── page.js                    # Главная страница со списком стран
│   ├── layout.js                  # Корневой layout (создан автоматически)
│   └── countries/
│       └── [id]/
│           ├── page.js            # Динамическая страница страны
│           └── cities/
│               └── [cityId]/
│                   └── page.js   # Динамическая страница города
├── data/
│   └── countries.js              # Многомерная структура данных
├── package.json
├── next.config.js
└── README.md
```

## Данные

Проект включает информацию о 5 странах:
- Россия (Москва, Санкт-Петербург, Новосибирск)
- США (Нью-Йорк, Лос-Анджелес, Чикаго)
- Франция (Париж, Марсель, Лион)
- Япония (Токио, Осака, Киото)
- Германия (Берлин, Мюнхен, Гамбург)

## Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone https://github.com/vsevolod6/next.jskt2.git
cd next.jskt2
```

2. Установите зависимости:
```bash
npm install
```

3. Запустите проект:
```bash
npm run dev
```

4. Откройте в браузере: http://localhost:3000

## Технологии

- **Next.js 14.0.0** - React фреймворк
- **React 18** - Библиотека для создания пользовательских интерфейсов
- **JavaScript ES6+** - Язык программирования

## Особенности реализации

- Использование App Router Next.js
- Динамические маршруты для стран и городов
- Многомерная структура данных с вспомогательными функциями
- Навигация между страницами с помощью компонента Link
- Обработка несуществующих маршрутов через notFound()
